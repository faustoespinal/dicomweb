Glossary

QIDO-RS Query based on ID for DICOM Objects
STOW-RS STore Over the Web
WADO-RS Web Access to DICOM Objects
UID Unique Identifier
Study An organizational unit representing a set of imaging events, normally related to a single encounter, and consists of a set of series
Series An organizational unit representing a single imaging event, and consists of a set of instances
Instance An organizational unit that typically representing a single image or a set of frames. It may contain a set of frames
Frame An organizational unit typically representing a single image in a multi-frame instance
