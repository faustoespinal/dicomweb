DICOM Web Services

This website describes in detail the DICOM Web Services, allowing you to unlock healthcare images RESTfully. There are three core services:


Query
Look for studies, series and instances using QIDO

Retrieve
Download specific images by reference with WADO

Store
Store specific instances using STOW

These services can be requested in either XML or JSON. Check out the Examples page to see these services in action.

Note: this site is meant for reference purposes only - please refer to the specifications for detailed information on the standard.

History

Digital Imaging and COmmunications in Medicine (DICOM) is the imaging standard for healthcare applications, founded in 1983 by the American College of Radiology (ACR) and National Electrical Manufacturers Association (NEMA). It describes both the format and the transport method of each image and how they are to be grouped (by series, by study, and by patient). For more information about DICOM, visit their website.
